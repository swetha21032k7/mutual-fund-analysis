# Mutual Fund Analysis

## Project Overview

This project focuses on analyzing mutual fund performance using historical datasets and live NAV data. The objective is to build an end-to-end analytics pipeline involving data ingestion, validation, transformation, visualization, and dashboard development.

## Day 1 Progress

* Project structure created
* Git repository initialized
* Required dependencies installed
* 10 CSV datasets ingested and profiled
* Live NAV data fetched using MFAPI
* Fund master dataset explored
* AMFI code validation completed
* Data quality assessment documented
